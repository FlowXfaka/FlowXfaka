说明
本项目已经内置了 Artisan 命令：
php artisan admin:reset-password

这个命令只能在服务器终端里执行，不提供公网“忘记密码”页面。

通用操作步骤

1.进入终端：
cd 你的项目目录

2. 执行重置命令
   php artisan admin:reset-password

3. 按提示操作
   - 如果只有一个管理员账号，会自动选中
   - 如果有多个管理员账号，会先让你选择要重置哪个管理员
   - 然后输入两次新密码

示例
cd /www/wwwroot/你的文件名
php artisan admin:reset-password

也可以直接指定管理员账号或邮箱
php artisan admin:reset-password 管理员账号
php artisan admin:reset-password 管理员邮箱

也可以直接带新密码
php artisan admin:reset-password 管理员账号 --password="你的新密码"

注意事项
1. 新密码至少需要 8 位。
2. 必须先进入项目根目录再执行命令。
3. 不同站点目录不同，不要把示例当成固定路径。
4. 如果系统里没有管理员账号，命令会直接报错提示。
5. 这个命令只重置管理员密码，不会影响前台用户数据。
